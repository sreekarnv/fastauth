# Installation

FastAuth can be installed via pip or Poetry.

## Requirements

- Python 3.11 or higher
- FastAPI 0.125.0 or higher

## Install from PyPI

```bash
pip install sreekarnv-fastauth
```

## Install with Poetry

```bash
poetry add sreekarnv-fastauth
```

## Verify Installation

```python
import fastauth
print(fastauth.__version__)
```
